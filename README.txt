────────────────────────────────
4. 배포 방법
────────────────────────────────
1) ZIP을 먼저 압축 해제합니다.
2) GitHub hihighportfolio 저장소에서 Add file → Upload files를 누릅니다.
3) index.html, styles.css, script.js, ui.js, README.txt, assets 폴더를 모두 드래그합니다.
4) Commit directly to the main branch를 선택하고 Commit changes를 누릅니다.
5) Vercel → hihighportfolio → Deployments에서 최신 배포가 Ready인지 확인합니다.
6) https://hihighportfolio.vercel.app/ 에서 강력 새로고침합니다.
   Mac: Command + Shift + R / Windows: Ctrl + F5

────────────────────────────────
5. 로컬 확인
────────────────────────────────
GLB는 file:// 방식에서 차단될 수 있으므로 로컬 서버로 확인합니다.

python3 -m http.server 8000
http://localhost:8000

Three.js는 jsDelivr CDN을 사용하므로 3D 인터랙션에는 인터넷 연결이 필요합니다.
본문과 이미지 갤러리는 CDN이 실패해도 표시됩니다.

6. 시각화 강화 업데이트 (A~E)
────────────────────────────────
파일 추가: motion.js  ← index.html에 이미 연결되어 있습니다. 반드시 함께 업로드하세요.

[A] 섹션 배경 반전
· About / Contact = 어두운 전체폭 밴드, Proposal = 연한 톤 밴드
· 흰 배경만 이어지던 흐름에 명암 변화를 3번 만들어 스크롤 피로를 줄였습니다.

[B] 미디어 풀블리드
· 각 프로젝트 대표 영상이 카드 좌우 끝까지 확장됩니다.
· 본문 폭에 갇혀 있던 영상이 화면을 채우면서 시선이 멈추는 지점이 생깁니다.

[C] 수치 블록
· 프로젝트마다 큰 숫자 4개를 배치했습니다. 새로 만든 정보가 아니라
  기존 본문에 이미 있던 사실만 뽑아 키운 것입니다.
  단조 1'40" / 70% / 4명 / 4개월
  하몽 41컷 / 2일 / 100% / 7종
  Summer 8씬 / 34컷 / 20초 / 2회

[D] 컬러 시스템 확장
· 프로젝트별 액센트(단조 브라운, 하몽 퍼플, Summer 틸)를
  수치·소제목 구분선·요약 카드 테두리·문서 버튼 호버까지 확장했습니다.

[E] 스크롤 인터랙션
· 상단 스크롤 진행 바
· 수치 카운트업 (화면에 들어올 때 1회)
· 대표 영상·현장 스틸 미세 패럴랙스
· 요약/수치/캐릭터 3단계/스킬/제안 카드 순차 등장
· prefers-reduced-motion 설정 시 모든 모션이 자동으로 꺼집니다.
